# Functionality
- The user is given a french word at the front of the card (white side)
- The user has 3 seconds to guess the english definition, after that, the card flips (blue side) to show the definition
- The user can click the bottom buttons depending on whether their guess was right or wrong

## Buttons
- After the user clicks any of the buttons, it will move to the next card
- If the user pressed the right button, a check mark will display or,
- if the user pressed the wrong button, a cross mark will display
- This keeps track of the number of right and wrong answers the user got
